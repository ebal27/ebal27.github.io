Cryptocurrency

Cryptocurrency is digital money you can only see on computers or phones. People use it to buy things, trade, or invest. One fun fact: the first famous Bitcoin purchase paid for two pizzas! Crypto became popular because it felt new and gave people a way to trade outside regular banks. It also shows how humans can decide that invisible numbers are worth something — we agree on value together.

The Internet

The internet is a gigantic network of computing devices that lets data move from one place to another — so you can message friends, stream videos, and learn stuff.

Vocab (4 selected)

  • Computing Device: A physical device that can run a program (e.g., computer, smartphone).
  • Computer Network: A group of interconnected computing devices capable of sending or receiving data.
  • Protocol: An agreed set of rules that specify how systems behave and communicate.
  • Bandwidth: The maximum transmission capacity of a system measured in bits per second.
